# Cyber Store - Full-Stack E-commerce Platform

## Giới thiệu

Cyber Store là một nền tảng thương mại điện tử được thiết kế và xây dựng để cung cấp trải nghiệm mua sắm trực tuyến hiện đại. Hệ thống hỗ trợ quản lý sản phẩm, giỏ hàng, đơn hàng, người dùng, và blog, với giao diện thân thiện và hiệu suất cao.

---

## Tính năng chính

### **Khách hàng**
- Đăng ký, đăng nhập và quản lý tài khoản cá nhân.
- Duyệt danh mục sản phẩm, tìm kiếm và lọc sản phẩm.
- Thêm sản phẩm vào giỏ hàng và wishlist.
- Đặt hàng và xem lịch sử đơn hàng.
- Thanh toán hóa đơn (hỗ trợ thanh toán trực tuyến nếu có).
- Đọc blog và gửi liên hệ đến quản trị viên.

### **Quản trị viên**
- Quản lý sản phẩm: thêm/sửa/xóa sản phẩm, danh mục, và hình ảnh.
- Quản lý người dùng: phân quyền, khóa/mở khóa tài khoản.
- Quản lý đơn hàng: cập nhật trạng thái đơn hàng.
- Quản lý blog: thêm/sửa/xóa bài viết và danh mục blog.
- Quản lý liên hệ: xử lý các yêu cầu từ khách hàng.

---

## Công nghệ sử dụng

### **Frontend**
- **React**: Thư viện xây dựng giao diện người dùng.
- **TypeScript**: Ngôn ngữ lập trình tĩnh, giúp giảm lỗi và tăng hiệu quả phát triển.
- **TailwindCSS**: Framework CSS để xây dựng giao diện nhanh chóng.
- **Vite**: Công cụ build hiện đại, tối ưu hóa hiệu suất.

### **Backend**
- **Node.js**: Nền tảng phát triển server-side.
- **Express.js**: Framework xây dựng API RESTful.
- **Prisma ORM**: Quản lý cơ sở dữ liệu MySQL.
- **Cloudinary**: Lưu trữ và quản lý hình ảnh.
- **JWT**: Xác thực người dùng.
