# Frontend - Electronics Store

## Cài đặt

```bash
npm install
```

## Chạy development server

```bash
npm run dev
```

Server sẽ chạy tại: http://localhost:5173

## Build production

```bash
npm run build
```

## Lưu ý

- Đảm bảo backend đang chạy tại http://localhost:4000
- Nếu gặp lỗi, hãy xóa `node_modules` và chạy lại `npm install`
